# Бот часть
## API  
> dbfunc - управление БД  
> parse_smiles - создание файла с кодировками смайлов  
> libvirt_api - управление ВМ  

## Запуск  
> `pip install -r requirements.txt`  
> настройте файл config.py  
> `python3 main.py`

# Сайт часть  
## Запуск  
> `pip install -r requirements.txt`  
> `python -m flask run`
